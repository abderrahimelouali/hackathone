# Experiencia Tinghir - Oasis Trails Connect

A modern, interactive tourism platform designed to showcase the authentic beauty and culture of the Tinghir region, Morocco.

## Features
- **Interactive Map**: Explore the Todra Gorge, Oasis, and historical sites with a custom MapLibre-driven experience.
- **Authentic Activities**: Book unique local experiences from traditional cooking to mountain climbing.
- **Local Marketplace**: Purchase authentic Moroccan products like Saffron, Berber carpets, and artisanal pottery.
- **Bilingual Interface**: Supporting Arabic, French, and English for a global audience.
- **Modern UI**: Built with Shadcn/UI, Tailwind CSS, and Framer Motion for a premium look and feel.

## Tech Stack
- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS
- **Components**: Shadcn/UI, Lucide React
- **Mapping**: MapLibre GL JS, MapCN
- **State Management**: React Context, TanStack Query
- **Animations**: Framer Motion

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the development server.

## Purpose
This project aims to connect travelers with authentic local hosts and artisans in the Tinghir region, promoting sustainable and culturally respectful tourism.
